FAQ

Respostas diretas para decisões rápidas.

O bot sai do servidor se eu não renovar?

Não. O Nexor permanece no servidor. Apenas recursos premium ficam bloqueados (C1).

Como funciona o mês calendário?

Renova no mesmo dia do mês seguinte. Se o dia não existir (ex.: 31), o vencimento vai para o último dia do mês.

Como a ativação funciona sem webhook?

Você gera o Pix dentro do Discord. Após pagar, o Nexor valida por consulta (polling) no pagamento.

O que eu preciso para ativar?

Adicionar o bot no servidor e usar o painel Premium. O pagamento é Pix de R$ 9,90.
